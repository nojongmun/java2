1. Ex01.java 파일 해석 하기
  => class Ex01 {   }
  => 저장할때 Ex01.java로 반드시 저장해야 된다. 
  => 저장이름 = 클래스이름 = 파일이름  
  => 클래스이름 무조건 첫글자가 대문자이어야 한다.
     특수문자, 띄워쓰기 사용 안됨

2. 클래스
   => 자바에서 파일단위 저장하는 단위
   => 자바는 클래스를 이용해서 객체를 만들고
      객체를 사용하는 언어
   => 자바 API : 회사에서 제공하는 클래스들의 모임
   => 자바 JDK 설치하면 자바API와 JVM이 설치된다.
   => 상태값(변수,상수)과 기능(메소드)을 가지고 있다.
   => 상태값을 Field 라고 한다.
   => 기능을 Method 라고 한다.



3. System.out.println("자바의 꿈"); 대한 해석을 하자
  => System 클래스를 API에서 찾아서 분석하자
   - 즐겨찾기에서 API 실행
   - 패키지 창에서 All Classes를 클릭하면 모든 클래스를 찾을 수 있다.
   - ctrl + f => 찾기 대화상자
   - 찾기 대화상자에서 원하는 클래스이름을 입력하여서 찾는다.
   - 찾은 클래스를 클릭하면 설명창에 해당 클래스 내용 나온다. 
    -- 상태값을 Field 라고 한다.
    -- 기능을 Method 라고 한다.
   - out은 The "standard" output stream.
    -- 표준 출력장치를 뜻한다. (모니터)
   - in은 The "standard" input stream.
    -- 표준 입력장치를 뜻한다. (키보드)
   - 
   - println()는 메소드이고 
     메소드란 기능, 동작, 실행하는 것을 말한다.
   - ()가 있으면 무조건 메소드 이다.
 
   - println()는 표시해라, 출력해라 동작을 가지고 있음 
   - ; 은 구문의 끝 표시(반드시 해야 되는 표시),  

4. public static void main(String[] args) {  }
   - main() : 메인메소드라고 한다.
   - 메인메소드는 자바프로그램의 시작위치
   - 메인메소드는 JVM(해석, 실행)이 호출하는 메소드
   - 메인메소드가 없어도 안되고, 2개이상 존재해도 안됨
